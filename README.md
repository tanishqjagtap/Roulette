# 🎰 Unity Slot Machine Game

## 📌 Game Overview
This project is a playable slot machine game built using Unity.

The game features:
- Smooth spinning reel animations
- Randomized slot outcomes
- Betting system
- Win and payout logic
- Clean UI and symbol alignment
- WebGL browser support

---

## 🎮 How to Play
1. Enter a bet amount
2. Pull the lever / press spin
3. Match symbols to win money

### Winning Rules
- 2 matching symbols = 2x payout
- 3 matching symbols = 3x payout (Jackpot)
- All different symbols = Lose bet

## 🧠 Approach / Thought Process
The project was designed using an object-oriented structure with separate scripts for:
- Reel spinning logic
- Slot machine game management
- Win condition handling

The main focus was creating smooth reel animations, randomized outcomes, and a clean user experience similar to classic slot machine games.
